# Getting start

## step1: install requirements

```base
pip install -r ./requirements.txt
```

## step2: run project:

### using python

```base
python .\manage.py runserver
```

### using py

```base
py .\manage.py runserver
```

if you have problem with installation , try to run => python -m pip install --upgrade pip
